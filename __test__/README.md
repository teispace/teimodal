# Documentation for the tests

## How to run the tests

### Prerequisites

- [Node.js](https://nodejs.org/en/) (version 18 or higher)

- Configure Jest

```bash
npx jest --init
```

### Running the tests (all)

```bash
npm test
```

### Running the tests (one)

```bash
npm test -- <path-to-test-file>

# Example
npm test -- app.test.js
```
